# Rubinization Guide: Adding Realistic Observational Effects

## Overview

Rubinization is the process of adding realistic observational effects to perfect (noise-free) mock lensed images, making them look like actual Rubin LSST observations. This guide explains how to use the Rubinization modules in both branches of AGENT_sim.

## What is Rubinization?

Perfect mock images generated by lensing simulations lack realistic observational effects. Rubinization adds:

1. **PSF (Point Spread Function)** - Blurs point sources to realistic sizes
2. **Photon Noise** - Random Poisson noise from discrete photons
3. **Read Noise** - Detector electronics noise
4. **Sky Background** - Thermal and scattered light from the sky
5. **Cosmic Rays** - High-energy particles hitting the detector
6. **Saturation** - Pixels that overflow the detector's dynamic range

## Quick Start

### PyLensLib+ImSim Branch

```python
from agent_sim.pylenslib_imsim_branch.imaging import ImSimRubinizer
import numpy as np

# Create Rubinizer with Rubin LSST parameters
rubinizer = ImSimRubinizer(
    psf_sigma=0.7,          # arcsec (Rubin design target)
    read_noise_e=10,        # electrons
    sky_brightness_mag=22,  # mag/arcsec^2
    gain=3.0                # e-/ADU
)

# Rubinize a perfect image
perfect_image = your_perfect_mock_image  # numpy array
rubinized_image, psf = rubinizer.rubinize(
    perfect_image,
    pixel_scale=0.2,        # arcsec/pixel
    add_psf=True,
    add_noise_flag=True,
    add_cr=True             # cosmic rays
)

# Output is in ADU (Analog-to-Digital Units)
print(f"Rubinized image shape: {rubinized_image.shape}")
print(f"Pixel value range: {rubinized_image.min():.1f} - {rubinized_image.max():.1f} ADU")
```

## Rubin LSST Parameters

All Rubinizers use LSST-compliant defaults:

| Parameter | Default | Unit | Justification |
|-----------|---------|------|---------------|
| `psf_sigma` | 0.7 | arcsec | LSST camera design target |
| `read_noise_e` | 10 | electrons | Segment-level electronics spec |
| `sky_brightness_mag` | 22 | mag/arcsec² | Dark site + dark gray twilight |
| `gain` | 3.0 | e-/ADU | Segment specification |
| Saturation level | 65,000 | ADU | Full-well capacity |
| Cosmic ray rate | 0.1% | pixels | Low Earth orbit flux |

## Understanding the Pipeline

### 1. PSF Convolution

The PSF blurs the perfect image to match atmospheric + instrumental resolution:

```python
# Create PSF kernel (Gaussian with FWHM = 2.355 * sigma)
psf = rubinizer.create_psf_kernel(pixel_scale=0.2, size=31)

# PSF is normalized so sum = 1.0
print(f"PSF sum: {psf.sum():.6f}")  # ~1.0

# Apply PSF to image
convolved = rubinizer.apply_psf(perfect_image, psf)
```

### 2. Noise Addition

Three types of noise are added:

**Sky Noise (Poisson)**: Random fluctuations from sky photons
- Magnitude: sqrt(sky_photons_per_pixel) electrons

**Photon Noise (Poisson)**: Random fluctuations from source photons  
- Magnitude: sqrt(source_photons) electrons

**Read Noise (Gaussian)**: Thermal noise from readout electronics
- Magnitude: read_noise_e electrons (default 10 e-)

```python
# Example with sky_brightness_mag = 22 mag/arcsec²
# At pixel_scale = 0.2 arcsec/pixel (0.04 arcsec²/pixel):
# Flux per pixel ~ 10^(-22/2.5) * 0.04 ≈ 1e-5 (in normalized units)
# Converted to electrons: ~ 100 e- per pixel per exposure
# Poisson noise: sqrt(100) ≈ 10 e- per pixel (5% noise)

# Plus read noise: 10 e- per pixel
# Total: ~14 e- per pixel noise
```

### 3. Cosmic Rays

Simulated as random hits that create streak patterns:

```python
# Cosmic ray parameters
cr_rate = 0.001         # 0.1% of pixels hit
cr_strength = 100       # electrons per hit

# Each cosmic ray creates a ~3x3 kernel streak
# Realistic for low Earth orbit (~100 cm²/s cosmic ray flux)
```

### 4. Saturation

Pixels exceeding the detector's full-well capacity are clipped:

```python
# Saturation level (realistic for LSST)
max_adu = 65000  # Full-well capacity

# Pixels > max_adu are clipped to max_adu
saturated_pixels = np.where(image > max_adu)
```

## Step-by-Step Example

### Full Rubinization Pipeline

```python
import numpy as np
from agent_sim.pylenslib_imsim_branch.imaging import ImSimRubinizer
from agent_sim.core.models import NFW, Cluster
from agent_sim.pylenslib_imsim_branch import PyLenslibSimulator

# Step 1: Create cluster and compute lensing
cluster = Cluster(NFW(M200=1e15, c=4.0, redshift=0.3))
simulator = PyLenslibSimulator(cluster)
lensing_maps = simulator.compute_lensing_maps(
    x_range=(-100, 100), y_range=(-100, 100), n_pixels=128
)

# Step 2: Generate mock images
from agent_sim.pylenslib_imsim_branch.imaging import SourcePlacement
sources = [
    SourcePlacement(
        source_id=0,
        source_position=(-0.5, 0.3),
        redshift=1.0,
        magnitude=20,
        radius_arcsec=0.1
    )
]

lensed_images, combined_mock = simulator.generate_mock_images(
    sources, x_range=(-100, 100), y_range=(-100, 100), 
    image_size=256, pixel_scale=0.2
)

# Step 3: Rubinize the mock image
rubinizer = ImSimRubinizer(psf_sigma=0.7, read_noise_e=10,
                           sky_brightness_mag=22, gain=3.0)

# Use random seed for reproducibility
np.random.seed(42)
rubinized_image, psf_kernel = rubinizer.rubinize(
    combined_mock,
    pixel_scale=0.2,
    add_psf=True,
    add_noise_flag=True,
    add_cr=True
)

# Step 4: Analyze results
print(f"Perfect image: {combined_mock.min():.6f} - {combined_mock.max():.6f}")
print(f"Rubinized image: {rubinized_image.min():.1f} - {rubinized_image.max():.1f} ADU")
print(f"Image std dev increased: {combined_mock.std():.6f} → {rubinized_image.std():.1f} ADU")
```

## Interpreting Output

### ADU Values

Rubinized images are in **ADU (Analog-to-Digital Units)**:
- 1 ADU = 3 electrons (for Rubin, gain = 3 e-/ADU)
- Typical range: 0-65,000 ADU (saturates above)
- Sky background: ~300-500 ADU (from 22 mag/arcsec² sky)

### Noise Estimates

Rough noise budget at sky_mag = 22:
```
Total noise ≈ sqrt(sky_noise² + photon_noise² + read_noise²)
            ≈ sqrt((10)² + (source_photons) + (3.3)²)
            ≈ 10-20 ADU per pixel (for faint sources)
```

## Comparison: Before and After

| Metric | Perfect | Rubinized |
|--------|---------|-----------|
| Smooth ring | Yes | Blurred by PSF |
| Zero background | Yes | ~400 ADU sky |
| No noise | Yes | ±10-20 ADU noise |

## FAQ

**Q: Why are my Rubinized images different each time?**
A: Noise is randomly sampled. Set `np.random.seed()` before calling `rubinize()` for reproducibility.

**Q: Should I Rubinize before or after combining multiple images?**
A: Typically you Rubinize individual exposures, then combine them. Each exposure should have independent noise.

**Q: How do I match the Rubin LSST observing conditions?**
A: Use the default parameters. Adjust `sky_brightness_mag` for different twilight levels (19-22).

**Q: Can I use a custom PSF?**
A: Yes, create your own PSF kernel and pass it to `apply_psf()`.

**Q: What's the output format?**
A: 2D numpy array in ADU. Can be saved with `np.save()` or `np.savez()`.

**Q: How realistic is the noise?**
A: Very realistic for standard Rubin observations. Matches Rubin system specs and typical sky background.

## Technical Details

### PSF Implementation

Uses Gaussian PSF with FWHM = 2.355σ:
```
PSF(x,y) = (1/(2πσ²)) * exp(-(x²+y²)/(2σ²))
```

### Noise Implementation

1. **Sky noise**: `np.random.poisson(sky_flux)`
2. **Photon noise**: `np.random.poisson(source_flux) - source_flux`
3. **Read noise**: `np.random.normal(0, read_noise_e)`

### Cosmic Ray Implementation

Randomly selected pixels get increased values via 3×3 kernel:
```
CR_kernel = [[0.1, 0.3, 0.1],
             [0.3, 1.0, 0.3],
             [0.1, 0.3, 0.1]]
```
